# Junior
First code from basic course
На вход подаются целых три числа - день, месяц и год (по григорианскому календарю). Выведите true, если эти числа составляют валидную дату, и false - в противном случае.
Sample Input 1:
25 8 2010
Sample Output 1:
true
Sample Input 2:
1 1 1
Sample Output 2:
true
